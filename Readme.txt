A savoir

Si l'installation via Dreamweaver n'est pa lié à un repo distant, via le terminal git que l'on lance via la vue fichier de Dreamweaver, la commande git init permet de créer un répertoire .git, le fchier .gitignore liste les directory comme les packages nodes à en laisser en local ensuite il faut lancer les commandes suivantes

1: git init
2: git add *
3: git status
4: git commit -m "Premier commit"

pour la première liasion ensuite passer cette commande
git remote add origin https://github.com/lerhun22/sass-grunt.git

5: git push -u origin master
